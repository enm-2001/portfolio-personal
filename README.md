# 🌐 Ekta Mer - Portfolio Website

A modern and responsive personal portfolio website built using **React** and **Tailwind CSS**. It showcases my projects, skills, and experience as a Software Developer.

---

## 🚀 Live Demo

👉 https://your-portfolio.vercel.app

---

## ⚙️ Tech Stack

- React
- Tailwind CSS
- Vite
- EmailJS (Contact Form)
- Lucide Icons

---

## ✨ Features

- 💻 Responsive design for all devices  
- 🌙 Clean modern UI  
- 📂 Featured projects section  
- 🧠 Skills section with categories  
- 📩 Contact form with email integration  
- 📄 Resume download option  
- 🔗 Social media links  

---

## 🛠️ Installation & Setup

git clone https://github.com/enm-2001/portfolio.git
cd portfolio
npm install
npm run dev

Open in browser:

http://localhost:5173
